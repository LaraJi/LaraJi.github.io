Trojaning Attack on Neural Networks

Posted on 2018-10-26 | Post modified: 2018-10-29 | In Paper | |
Words count in article: 196 word | Reading time ≈ 1 min

Paper Loc

Trojan attack, craft input-agnostic mask and tune model to inject the trojan trigger into the model. The generated model can retain accuracy on normal inputs, but will misclassify the inputs with trojan trigger(the crafted mask) as the trojan target label.

Method

Generate trojan mask

  1. Initialize mask: with a specific shape which is easy for mask optimization;
  2. Select nerons to be activated: strong-connected neurons;
  3. Optimize mask: backpropagate to make neurons largest activated.

Generate training data

  1. Average all images of a certain dataset (don’t have to be the original dataset of the original model);
  2. Craft class-representative images for each class: backpropagate to make model classification probability for each class close to 1;
  3. Denoise: make crafted images more nature, and further improve retraining accuracy.

Retrain trojan model

  1. Train model layers after the modified layer;
  2. Tarining-data includes crafted images with/without the mask with its original/targeted label.

Future Work

Check the prediction class distribution –> trojan model tends to predict high percent of inputs to a target label.

Possible Extensions

  1. Will model compression help defense? –> fine-pruning paper
  2. Interpretation works for defense?
